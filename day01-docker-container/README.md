# Day 01 - Docker Container Creation

## 📌 What I did
- Created a simple Python application
- Wrote a Dockerfile
- Built a Docker image
- Ran a Docker container

## ⚙️ Commands used

docker build -t myapp .
docker run myapp

## 🧠 What I understood
- Image is a prepared environment
- Container is a running instance of an image
- Docker avoids manual dependency setup

## 🚀 Output
Hello from my container
